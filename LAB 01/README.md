# Lab 01 – Personal Portfolio Website

## Overview

This project is created as **Lab 01** for the **Web Development–I** course of **B.Tech CSE** at **K.R. Mangalam University**. It is a simple personal portfolio website built using **HTML** to practice basic web development concepts.

## Technologies Used

* HTML5

## Features

* Portfolio layout with sections: About, Projects, Skills, and Contact
* Use of images, lists, tables, and forms
* Simple navigation menu

## How to Run

Open the `index.html` file in any web browser.
