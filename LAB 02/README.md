# Personal Portfolio Website

## Overview
This is a simple personal portfolio website created to practice front-end web development basics. It presents personal information, skills, sample projects, and a contact form in a clean and structured layout.

## Technologies Used
- HTML5
- CSS3

## Features
- Responsive page structure
- About section with profile image
- Skills table
- Projects showcase
- Contact form
- Clean and minimal UI design

## How to Run
Open the `index.html` file in any modern web browser.
